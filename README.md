# Retail-waste-profit-optimization
Business analytics and machine learning project focused on reducing retail inventory waste and improving profitability using EDA, visualization, and predictive modeling.

**Project Overview**

This project analyzes retail transaction data to identify key drivers of inventory waste and profitability. Using exploratory data analysis, visualization, and predictive modeling, the project provides business recommendations for reducing waste and improving operational efficiency.

**Business Problem**

Retail businesses lose significant revenue due to:
perishable inventory waste
poor pricing strategies
inefficient promotions
inaccurate inventory planning

The objective was to identify the operational and pricing factors contributing to waste and profit decline.

**Dataset Information**

100,000 transaction records
41 business variables
50 stores across 5 regions
63 unique products
Time Period: 2023–2024

Dataset includes:

inventory
pricing
promotions
demand
spoilage
revenue
profit metrics

Key Insights
Waste Drivers
High waste strongly linked with low sales velocity
Frozen meals showed highest waste percentage
Aggressive discounting reduced profitability
Profitability Findings
Profit declines significantly after high discount thresholds
Inventory freshness directly affects margin performance

**Business Recommendations**

Implement dynamic markdown pricing
Reduce overstocking for low-velocity products
Use category-specific inventory policies
Build waste-risk monitoring dashboards
